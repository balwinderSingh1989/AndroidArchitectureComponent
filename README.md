Android Architecture Components Sample -

A Sample project of the news app that uses ViewModels, LiveData, Locale Cache with Room and Data Binding with an MVVM architecture.

- Room
- LiveData
- Data Binding
- RxJava2

- OkHttp3
- Retrofit2
- ViewModels
